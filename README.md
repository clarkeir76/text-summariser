# Text Summariser

A simple text summariser tool that takes a body of text and produces a concise summary.

## Overview

This project provides a straightforward way to summarise long-form text into shorter, digestible output — useful for quickly understanding the key points of articles, documents, or any large block of text.

## Features

- Input any block of text
- Receive a concise, readable summary
- Simple and lightweight
